Weitere Anmerkungen:
All photos and icons were generated by AI
The prototype does not include a backend, so data is not stored persistently.
All navigation and user flow are handled locally within the app.
Progress is shown on the homepage, but resets after restarting the app.
A side drawer menu is implemented with dummy buttons: “Settings” and “About this app”.
The login screen simulates user access; there is no real authentication.
Image loading may take longer than expected depending on the device and connection.
The logout button under the avatar redirects the user to the Authorization Screen.
The avatar buttoun redicters users to their profile.
